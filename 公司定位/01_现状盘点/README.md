# 现状盘点

本页记录当前已经基本确认的事实和表达边界，避免后续讨论把“品牌主张”“客群话术”“公司类型”混在一起。

## 已确认信息

- 公司名：`LinearVision`
- 品牌主张：`See. Think. Act. With AI.`
- 中文辅助表达：`看见、思考、行动，AI 与你并肩`
- 当前品牌展开方式：围绕 `See / Think / Act` 三段式展开产品、服务和 landing page
- 当前服务对象：商业客户，优先面向银行与金融机构、投资机构、科技公司、创业团队
- 当前品牌阶段：市场对 `LinearVision` 还没有明确认知，仍然需要 `With AI` 帮助陌生客户快速建立品类认知

## 当前明确产品线

- `AI Agent绩效`：面向企业已经部署多个 Agent 后，对绩效、可观测性、责任归属和 ROI 管理的需求
- `AI安全`：面向企业在 Agent 权限、审计、合规、动作验证和风险控制上的治理需求

这两条产品线都不是脱离定位单独存在的产品，而是 `See / Think / Act` 架构在企业真实场景里的第一批具体落点。

## 当前定位边界

- 我们不是基础模型研发公司。
- 我们不主打“自研大模型”，而是主打“把顶尖大模型变成业务能力”。
- 我们的价值不在底层模型本身，而在于理解业务场景，把 AI 嵌入到观察、思考和行动链路里。
- 我们不应该把自己做成卖工时、卖人头的 AI 外包公司。

## 当前更适合使用的公司定义

- `AI 应用方案商`
- `AI 决策与执行系统构建者`
- `基于顶尖大模型的业务落地伙伴`
- `连接信息、判断与行动的 AI 系统提供方`

## 当前阶段判断

- 现阶段可以说自己是客户的 `业务伙伴` 或 `关键系统层`，但还不适合过早自称 `企业操作系统`。
- 现阶段要强调 `可信、落地、业务结果`，而不是强调宏大愿景。
- 现阶段真正缺的不是新 tagline，而是让 `See / Think / Act` 对应真实能力和案例。

## 当前暂不建议强调的方向

- 强调自己是基础模型公司
- 过早放大“自研模型”叙事
- 只讲技术架构，不讲客户结果
- 在没有足够产品和客户支撑前，直接说自己是 `Foundational Software` 或 `企业操作系统`
