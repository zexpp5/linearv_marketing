# 企业智能客服

> 接微信/企微/网站，AI 自动应答 + 人工兜底，替代 3-5 个初级客服。

## 业务痛点

| 痛点 | 具体表现 |
|------|----------|
| 人力成本高 | 初级客服月薪 4000-6000，加上社保管理成本，5 人客服团队年成本 40-50 万 |
| 响应慢/漏回 | 非工作时间无人值守，客户等 10 分钟以上流失率超 60% |
| 质量不稳定 | 不同客服回答口径不一致，新人培训周期 2-4 周 |
| 知识更新难 | 产品更新后客服话术跟不上，FAQ 文档散落各处 |
| 数据沉没 | 大量对话记录没人看，客户高频问题和情绪变化无人分析 |
| 多渠道割裂 | 微信、企微、网站、小程序各一套系统，数据不互通 |

## 核心功能

- 多渠道接入（微信公众号、企业微信、网站在线客服、小程序）
- 基于企业知识库的智能应答
- 意图识别 + 自动分流（简单问题 AI 答，复杂问题转人工）
- 对话记录自动总结与工单生成
- 客户情绪识别与预警

## 目标客户

- 电商/零售（售前咨询 + 售后处理）
- 教育机构（课程咨询 + 学员服务）
- SaaS 公司（产品使用问答）
- 任何日均咨询量 50+ 的中小企业

### 我们的差异化机会

- **竞品问题**：头部产品偏重大中型客户，定价对中小企业仍然偏高；功能大而全但上手复杂
- **我们的切入点**：
  - 轻量化部署，聚焦"知识库驱动的问答"这一核心能力
  - 与方向 A01（数据服务）联动，帮客户从 0 搭建知识库
  - 与方向 A05（托管运营）联动，持续优化回答质量
