# 企业智能客服

> 接微信/企微/网站，AI 自动应答 + 人工兜底，替代 3-5 个初级客服。

## 核心功能

- 多渠道接入（微信公众号、企业微信、网站在线客服）
- 基于企业知识库的智能应答
- 意图识别 + 自动分流（简单问题 AI 答，复杂问题转人工）
- 对话记录自动总结与工单生成
- 客户情绪识别与预警

## 目标客户

- 电商/零售（售前咨询 + 售后处理）
- 教育机构（课程咨询 + 学员服务）
- SaaS 公司（产品使用问答）
- 任何日均咨询量 50+ 的中小企业

## 交付物

- 部署完成的智能客服系统
- 企业知识库（首次搭建）
- 话术模板库
- 管理后台（对话监控、数据统计）

## 商业模型

| 维度 | 说明 |
|------|------|
| 客单价 | 3-8 万（部署）+ 3000-8000 元/月（运维） |
| 交付周期 | 2-4 周 |
| 核心卖点 | 省人力成本，7x24 在线，响应速度秒级 |
