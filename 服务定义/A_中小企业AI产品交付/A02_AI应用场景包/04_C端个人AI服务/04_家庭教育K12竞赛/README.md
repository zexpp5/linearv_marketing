## 3. 私域竞品调研方法

> 网上搜索只能找到公开产品。真正的竞品藏在小红书、闲鱼、微信生态和垂直论坛里，必须手动潜入。

### 小红书调研

**搜索关键词**（App 内搜）：
```
竞赛规划 / 白名单赛事 / 信息学竞赛 妈妈
竞赛日历 / 适合孩子的竞赛 / 竞赛怎么选
CSP 备赛 / 数学竞赛 规划 / 强基计划 竞赛
```

**关注**：
- 哪些博主在做竞赛类内容？粉丝量？
- 评论区有没有人问"有没有工具/日历"？
- 有没有挂商品链接卖"竞赛规划咨询"？
- 私信博主问"有没有付费社群"

### 闲鱼调研

**搜索关键词**：
```
白名单赛事资料 / 竞赛规划 / 信息学竞赛教程
CSP 真题 / 竞赛升学咨询 / 竞赛培训资料
```

**关注**：价格、销量（想要数）、是否有人卖咨询服务

### 微信生态调研

1. 微信搜一搜输入"白名单赛事"/"竞赛规划" → 看公众号
2. 关注排名前 5-10 个公众号，观察一周
3. 看菜单栏有没有小程序/付费课程/社群入口
4. 加入免费群观察变现模式
5. 微信小程序搜"竞赛"/"赛事" → 检查工具

### 知识星球

- 搜"竞赛"/"信息学"/"升学规划"
- 看人数、定价、更新频率
- 花 50-100 元加 1-2 个看内容

### 垂直论坛

| 论坛 | 搜什么 |
|------|--------|
| 家长帮（jzb.com） | "竞赛规划"/"白名单" |
| 洛谷（luogu.com.cn） | 信息学社区，看备赛讨论 |
| 知乎 | "竞赛规划 工具"/"白名单赛事 怎么选" |

### 调研执行计划（1 周）

```
Day 1-2: 小红书搜索 + 关注 10 个竞赛博主
Day 3:   闲鱼搜竞赛资料，记录品类/价格/销量
Day 4:   微信搜一搜 + 公众号 + 小程序
Day 5:   加入 1-2 个知识星球/付费社群
Day 6:   家长帮/洛谷/知乎浏览
Day 7:   汇总输出竞品表格
```
