# 04 C 端个人 AI 服务

> 纯线上/系统交付的个人付费 AI 服务。不涉及任何线下代办跑腿。

## 原则

- **只做线上能闭环的** — AI 生成、批改、匹配、咨询、内容生产
- **不做线下的** — 不跑腿、不代办、不上门
- 获客在小红书/闲鱼，成本极低
- 与 C01（自媒体 IP）天然联动
