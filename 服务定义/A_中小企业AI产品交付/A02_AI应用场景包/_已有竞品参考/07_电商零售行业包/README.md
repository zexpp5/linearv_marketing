# 电商/零售行业场景包

> 覆盖选品→上架→运营→售后全链路的 AI 能力包。

## 业务痛点

| 痛点 | 具体表现 |
|------|----------|
| 选品靠直觉 | 不知道什么品类有机会，跟风选品导致库存积压 |
| 上架效率低 | 多 SKU、多平台铺货，一个运营每天只能上 10-20 个品 |
| 舆情后知后觉 | 差评/投诉爆发后才发现，错过最佳处理窗口 |
| 售后工单积压 | 退货/换货/投诉分类全靠人，高峰期处理不过来 |
| 数据分散 | 各平台数据不互通，经营决策缺整体视角 |

## 场景清单

| 场景 | 说明 |
|------|------|
| 智能选品助手 | 竞品分析 + 趋势预测 + 利润测算 |
| 客户评价分析与舆情预警 | 多平台评论聚合分析，负面评价实时告警 |
| 自动化商品上架 | 多平台一键铺货 + AI 生成标题/详情页 |
| 退货/售后工单自动分类 | 工单自动分类 + 处理建议 + 优先级判定 |

## 目标客户

- 多平台经营的电商卖家
- 电商代运营公司
- 零售连锁品牌

### 我们的差异化机会

- **竞品问题**：选品工具、舆情工具、客服工具各自为政；卖家要同时用 3-5 个工具，数据不互通
- **我们的切入点**：
  - 打包"选品+上架+舆情+售后"一体化行业包
  - 与 05（电商运营助手）组合形成电商全链路 AI 解决方案
  - 以"评论分析+舆情预警"为钩子切入，再扩展全链路
