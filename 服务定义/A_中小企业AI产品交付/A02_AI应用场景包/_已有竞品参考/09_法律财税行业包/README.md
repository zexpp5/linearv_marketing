# 法律/财税行业场景包

> 合同审查、政策问答、票据处理，AI 替代高重复度的专业工作。

## 业务痛点

| 痛点 | 具体表现 |
|------|----------|
| 合同审查积压 | 中小律所/企业法务一人管上百份合同，漏审风险高 |
| 法规更新跟不上 | 财税政策频繁变更，会计/法务靠人肉追踪极易遗漏 |
| 发票处理人工重 | 每月几百张发票手工录入、核验、归类，财务月底加班常态 |
| 专业人才贵 | 中小企业请不起全职法务/高级会计，外包费用也不低 |
| 检索效率低 | 查一个法条/政策要翻多个网站和文件，30 分钟起步 |

## 场景清单

| 场景 | 说明 |
|------|------|
| 合同风险条款审查 | 自动标注风险点 + 修改建议（与 04 合同助手互补） |
| 财税政策问答助手 | 接最新政策库，自然语言查询 |
| 发票/报销单据自动识别与归类 | OCR + AI 自动提取信息并分类入账 |

## 目标客户

- 中小律所
- 财税代理公司
- 企业财务部门

### 我们的差异化机会

- **竞品问题**：法律 AI 偏向律所和大企业法务；财税 AI 偏向发票/账务处理，两者割裂
- **我们的切入点**：
  - 打包"法律+财税"一体化场景包，面向中小企业
  - 合同审查（复用 04 合同助手）+ 财税政策问答 + 发票处理三合一
  - 面向财税代理公司可做"代理公司的 AI 生产力工具"
