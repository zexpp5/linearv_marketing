# 合同/文档审阅助手

> AI 自动标注风险条款、对比模板差异、生成修改建议。

## 业务痛点

| 痛点 | 具体表现 |
|------|----------|
| 审阅效率低 | 一份 30 页合同人工审阅需 2-4 小时，法务积压严重 |
| 风险遗漏 | 非法律专业人员审合同，关键风险条款漏看（违约责任、管辖权、知识产权归属） |
| 版本混乱 | 合同来回修改 5-10 版，对比差异全靠肉眼 |
| 模板偏离 | 业务部门拿到对方合同直接签，未与企业标准模板比对 |
| 创业公司无法务 | 中小企业请不起全职法务，外聘律师审一份合同 2000-5000 元 |
| 历史合同难查 | 签过的合同找不到、查不了，纠纷时翻箱倒柜 |

## 核心功能

- 合同上传 → AI 自动识别关键条款
- 风险条款标注 + 严重等级评估
- 与企业标准模板自动对比差异
- 修改建议自动生成
- 审阅历史记录与版本追踪

## 目标客户

- 法务团队（每月审阅合同 20+ 份）
- 采购部门（供应商合同审核）
- 创业公司（没有法务但合同多）

### 我们的差异化机会

- **竞品问题**：头部产品偏向有法务团队的中大企业；中小企业觉得"我又不是律所，用不上"
- **我们的切入点**：
  - 面向**没有法务的中小企业**，降低使用门槛
  - 不做全功能法律平台，聚焦"上传合同→看风险→给建议"这一条线
  - 可作为方向 09（法律财税行业包）的核心组件
