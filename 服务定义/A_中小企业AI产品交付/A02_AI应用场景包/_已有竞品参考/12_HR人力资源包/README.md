# HR/人力资源场景包（通用）

> 覆盖招聘→入职→管理全链路的 AI 能力，跨行业适用。

## 业务痛点

| 痛点 | 具体表现 |
|------|----------|
| 招聘筛选效率低 | 一个岗位 200-500 份简历，HR 人工筛 2-3 天 |
| 面试标准化难 | 不同面试官问题和评分标准不一致，结果靠感觉 |
| 员工满意度靠猜 | 年度调研做完堆在那里，没人分析也没人跟进 |
| 培训效果不可见 | 花了钱做内训，不知道员工到底学到了什么 |
| 人才数据孤岛 | 招聘、考勤、绩效、培训各一套系统，数据不互通 |

## 场景清单

| 场景 | 说明 |
|------|------|
| 简历解析 + 岗位匹配度评分 | 批量解析简历，自动与 JD 匹配打分 |
| 面试题库自动生成 | 按 JD 定制面试题，分技术/行为/情景 |
| 员工满意度调研分析 | 问卷数据自动分析，生成洞察报告 |

## 目标客户

- 年招聘量 50+ 的企业
- 猎头公司
- HR SaaS 公司（集成合作）

- **竞品问题**：北森/Moka 是完整 HCM 平台，中小企业用不上也买不起；专项工具只解决面试单点

  - 轻量 HR AI 工具包：简历筛选 + 面试题 + 满意度分析三件套
