# 教育/培训行业场景包

> 从课程设计到学员服务，AI 提升教育机构运营效率。

## 业务痛点

| 痛点 | 具体表现 |
|------|----------|
| 课程设计耗时 | 一门新课从大纲到课件需 2-4 周，讲师把时间花在排版而非内容 |
| 课后答疑压力大 | 学员重复问同样的问题，老师 / 助教忙不过来 |
| 培训效果难量化 | "上完课感觉不错"，但无法衡量知识吸收率和行为改变 |
| 内容更新慢 | 行业知识迭代快，课件跟不上最新变化 |
| 个性化不足 | 所有学员用同一套内容，基础差异大导致效果参差 |

## 场景清单

| 场景 | 说明 |
|------|------|
| AI 课程大纲生成器 | 输入主题自动出教学设计、课时安排、作业设计 |
| 学员问答助手 | 课后自动答疑，基于课程内容知识库 |
| 培训效果评估报告 | 自动生成学员表现分析、课程改进建议 |

## 目标客户

- 职业培训机构
- 企业内训部门
- 在线教育公司

### 我们的差异化机会

- **竞品问题**：K12 产品面向 C 端学生；企业培训平台偏重 LMS 管理，AI 能力是附加功能
- **我们的切入点**：
  - 面向**职业培训机构和企业内训**，不做 K12
  - 核心价值是"AI 课程生产工具"而非学习平台
  - 帮讲师快速生成课程大纲 + 课件 + 考题 + 评估报告
  - 可与 C02（咨询培训）自身业务复用，自己用的工具卖给同行
