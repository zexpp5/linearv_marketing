# 招聘筛选助手

> 简历解析 + 岗位匹配 + 面试题生成，HR 从海量简历中秒筛目标候选人。

## 业务痛点

| 痛点 | 具体表现 |
|------|----------|
| 简历筛选耗时 | 一个岗位收 200-500 份简历，HR 花 2-3 天初筛，仍靠主观判断 |
| 匹配度低 | 人工筛选漏掉好候选人，又面试了大量不匹配的人 |
| 面试标准不一 | 不同面试官问不同问题，评估标准主观 |
| 招聘周期长 | 从发布到入职平均 30-45 天，业务部门等不起 |
| 候选人体验差 | 投了简历没回复，面试后没反馈，雇主品牌受损 |
| 人才库浪费 | 历史简历堆在邮箱/ATS 里，新岗位不会去翻旧简历 |

## 核心功能

- 简历批量解析与结构化
- 岗位 JD ↔ 简历自动匹配评分
- 候选人背景自动调研
- 面试题库按 JD 自动生成
- 面试评估表自动生成

## 目标客户

- 月均招聘量 10+ 的中小企业
- 猎头/招聘外包公司
- 快速扩张期的创业公司

### 我们的差异化机会

- **竞品问题**：Moka/北森/猎聘面向中大型企业，年费门槛高；中小企业需要的是"简历筛选+面试题"而非全套 HCM
- **我们的切入点**：
  - 轻量级工具，聚焦"简历解析→匹配评分→面试题生成"三步
  - 可独立使用，也可作为插件对接客户现有 ATS
  - 按岗位/次数计费，降低中小企业使用门槛
