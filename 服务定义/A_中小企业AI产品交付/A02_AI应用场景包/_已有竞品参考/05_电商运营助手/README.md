# 电商运营助手

> 商品描述、客服话术、评论分析一站搞定，运营人效翻倍。

## 业务痛点

| 痛点 | 具体表现 |
|------|----------|
| 文案产量低 | 一个运营每天手写 5-10 条商品描述已是极限，上新旺季严重瓶颈 |
| 多平台适配累 | 同一商品在淘宝/京东/拼多多/抖音要写不同风格的标题和详情页 |
| 客服话术靠经验 | 新客服不知道怎么回答，老客服的好话术无法复制给团队 |
| 评论分析靠人看 | 上千条评论堆着没人分析，差评预警全靠客服偶然发现 |
| 竞品动态跟不上 | 竞品改了标题/价格/详情页，几天后才发现 |
| 上架效率低 | 多 SKU 上架填写字段耗时，人力成本高 |

## 核心功能

- 商品标题/详情页/卖点文案批量生成
- 客服话术库自动生成与优化
- 客户评论情感分析与关键词提取
- 竞品 listing 对比分析
- 多平台内容适配（淘宝/京东/拼多多/抖音）

## 目标客户

- 电商卖家（SKU 50+ 的中小卖家）
- 电商代运营公司
- 品牌方电商部门

### 我们的差异化机会

- **竞品问题**：阿里系工具绑定平台生态；第三方工具功能单一（只做铺货或只做文案）；没有"文案+话术+评论分析"一体化方案
- **我们的切入点**：
  - 跨平台一体化：同一内容自动适配淘宝/京东/拼多多/抖音
  - 聚焦"内容生产+评论洞察"闭环，不做铺货/供应链
  - 可与 07（电商零售行业包）组合，覆盖运营全链路
