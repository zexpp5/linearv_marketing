# 通用职能效率包（跨行业）

> 不分行业，任何公司都用得上的 AI 效率工具集。

## 业务痛点

| 痛点 | 具体表现 |
|------|----------|
| 会议开完就忘 | 没人记纪要，记了也没人看，待办事项无人跟进 |
| 周报月报靠编 | 每周花 30-60 分钟写周报，管理层汇总更久，信息失真 |
| 竞品动态靠偶遇 | 竞对发了新产品/改了价格，几周后才从客户嘴里听到 |
| SOP 查不到 | 公司有制度有流程，但员工不知道在哪找、懒得找 |
| 信息过载 | 每天海量消息/文档/邮件，真正重要的信息被淹没 |

## 场景清单

| 场景 | 说明 |
|------|------|
| 会议纪要自动生成 | 录音/文字 → 结构化纪要 + 待办提取 + 自动分发 |
| 周报/月报自动汇总 | 接项目管理工具（飞书/Jira/Notion），自动聚合生成 |
| 竞品监测与分析报告 | 定期抓取竞品动态，AI 生成分析报告 |
| 内部流程 SOP 问答助手 | 基于制度文档的智能问答，新人即查即用 |

## 目标客户

- 几乎所有 20 人以上的公司
- 特别适合作为**首单切入产品**，门槛低、需求普遍

### 我们的差异化机会

- **竞品问题**：
  - 大厂工具（飞书妙记/钉钉 AI）绑定各自生态，跨平台弱
  - 单点工具多（会议纪要一个、周报一个、SOP 一个），缺乏一体化
  - 竞品监测类工具偏贵，中小企业用不起
- **我们的切入点**：
  - 打包"会议纪要 + 周报 + SOP 问答 + 竞品监测"四合一
  - 不绑定飞书/钉钉，支持多平台
  - **最适合做首单切入产品**：需求普遍、决策门槛低、见效快
  - 从这里切入后可引导到 03（知识问答）或行业场景包
