# 电商运营助手

> 商品描述、客服话术、评论分析一站搞定，运营人效翻倍。

## 核心功能

- 商品标题/详情页/卖点文案批量生成
- 客服话术库自动生成与优化
- 客户评论情感分析与关键词提取
- 竞品 listing 对比分析
- 多平台内容适配（淘宝/京东/拼多多/抖音）

## 目标客户

- 电商卖家（SKU 50+ 的中小卖家）
- 电商代运营公司
- 品牌方电商部门

## 交付物

- 电商运营 AI 工具平台
- 行业话术模板库
- 评论分析看板
