# 01 招投标 AI

> 帮中小企业找标、写标、过合规 — 纯线上系统。

## 中小企业痛点

| 痛点 | 数据 |
|------|------|
| 找标难 | 招标平台碎片化，人工筛选耗时，错过商机 |
| 写标慢 | 一份标书平均 40-60 小时，无专职标书人员 |
| 废标率高 | 格式错误/资质遗漏导致废标率约 15% |
| 合规盲区 | 暗标格式新规严苛，细节偏差直接废标 |
