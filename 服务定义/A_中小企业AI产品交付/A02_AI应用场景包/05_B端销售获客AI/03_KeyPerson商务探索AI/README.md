# 03 Key Person 商务探索 AI

> 帮中小企业找到决策人、制定触达策略 — 纯线上系统。

## 中小企业 BD 痛点

| 痛点 | 具体表现 |
|------|----------|
| 找不到对的人 | 目标企业组织架构不透明，通讯录无从入手 |
| 决策链不清晰 | 不知道谁是真正决策者 vs 执行者 vs 影响者 |
| 触达方式单一 | 冷电话+群发邮件，转化率极低 |
| 信息碎片化 | 工商/招聘/社媒信息分散，无法综合判断 |
| 有效线索成本高 | 5 年内从 100 元/条涨至 500 元+（3 倍） |
