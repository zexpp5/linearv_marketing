# 招聘筛选助手

> 简历解析 + 岗位匹配 + 面试题生成，HR 从海量简历中秒筛目标候选人。

## 核心功能

- 简历批量解析与结构化
- 岗位 JD ↔ 简历自动匹配评分
- 候选人背景自动调研
- 面试题库按 JD 自动生成
- 面试评估表自动生成

## 目标客户

- 月均招聘量 10+ 的中小企业
- 猎头/招聘外包公司
- 快速扩张期的创业公司

## 交付物

- 招聘 AI 助手（Web 端或对接现有 ATS）
- 简历解析引擎
- 面试题库

## 商业模型

| 维度 | 说明 |
|------|------|
| 客单价 | 3-5 万（部署）+ 2000-4000 元/月 |
| 交付周期 | 2-3 周 |
| 核心卖点 | 筛选效率提升 10x，匹配准确率提升 |
