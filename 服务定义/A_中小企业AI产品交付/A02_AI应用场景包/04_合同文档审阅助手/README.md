# 合同/文档审阅助手

> AI 自动标注风险条款、对比模板差异、生成修改建议。

## 核心功能

- 合同上传 → AI 自动识别关键条款
- 风险条款标注 + 严重等级评估
- 与企业标准模板自动对比差异
- 修改建议自动生成
- 审阅历史记录与版本追踪

## 目标客户

- 法务团队（每月审阅合同 20+ 份）
- 采购部门（供应商合同审核）
- 创业公司（没有法务但合同多）

## 交付物

- 合同审阅系统（Web 端）
- 企业合同模板库
- 风险条款知识库（可持续扩充）

## 商业模型

| 维度 | 说明 |
|------|------|
| 客单价 | 5-10 万（部署）+ 3000-8000 元/月 |
| 交付周期 | 3-4 周 |
| 核心卖点 | 审阅效率提升 5-10x，降低漏审风险 |
