# 内部知识问答

> 接飞书/钉钉/语雀，员工问什么都能秒答，告别"翻文档找半天"。

## 核心功能

- 对接企业文档系统（飞书文档、语雀、Confluence、本地文件）
- 自然语言提问 → 精准回答 + 引用来源
- 权限管理（按部门/角色控制可访问知识范围）
- 知识库自动更新（文档变更自动同步）
- 高频问题统计与知识缺口分析

## 目标客户

- 30 人以上团队（内部文档开始难找）
- 新人入职频繁的快速扩张公司
- 有大量 SOP/制度文档的企业

## 交付物

- 知识问答机器人（飞书/钉钉/企微/Web）
- 知识库首次搭建与调优
- 管理后台（知识更新、问答统计）

## 商业模型

| 维度 | 说明 |
|------|------|
| 客单价 | 3-8 万（部署）+ 3000-6000 元/月 |
| 交付周期 | 2-4 周 |
| 核心卖点 | 新人上手快、老人不被打断、知识不随人走 |
