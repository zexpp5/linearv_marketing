# 方向 1：AI 应用场景包 / 行业解决方案

> 把技术栈封装成"解决某个具体岗位问题"的场景化产品。

## 产品形态

### 已有场景

- 企业智能客服（接微信/企微/网站）
- 销售线索助手（接 CRM，自动跟进/总结）
- 内部知识问答（接飞书/钉钉/语雀）
- 合同/文档审阅助手
- 电商运营助手（商品描述、客服话术、评论分析）
- 招聘筛选助手

### 拓展场景 — 按行业

**电商/零售**
- 智能选品助手（竞品分析 + 趋势预测 + 利润测算）
- 客户评价分析与舆情预警
- 自动化商品上架（多平台一键铺货 + AI 生成标题/详情页）
- 退货/售后工单自动分类与处理建议

**教育/培训**
- AI 课程大纲生成器（输入主题自动出教学设计）
- 学员问答助手（课后自动答疑）
- 培训效果评估报告自动生成

**法律/财税**
- 合同风险条款审查（标注风险点 + 修改建议）
- 财税政策问答助手（接最新政策库）
- 发票/报销单据自动识别与归类

**制造/供应链**
- 供应商资质自动审核
- 质检报告自动解读与异常预警
- 采购询价自动比对与汇总

**医疗/健康**
- 患者随访助手（自动外呼 + 记录 + 异常标记）
- 病历结构化与检索
- 医学文献摘要与问答

**人力资源（通用）**
- 简历解析 + 岗位匹配度评分
- 面试题库自动生成（按 JD 定制）
- 员工满意度调研分析

### 拓展场景 — 按职能（跨行业通用）

- 会议纪要自动生成与待办提取
- 周报/月报自动汇总（接项目管理工具）
- 竞品监测与分析报告
- 内部流程 SOP 问答助手

## 商业模型

| 维度 | 说明 |
|------|------|
| 客户买单逻辑 | 解决一个具体岗位/部门的问题，看得见 ROI |
| 典型客单价 | 3-10 万 / 项目 |
| 收费模式 | 一次性交付费 + 年度运维费 |
| 交付难度 | 中 |
| 现金流速度 | 中（成交周期 2-8 周） |

## 落地建议

- 优先选 2-3 个场景做成**可演示的标准 demo**，降低客户决策成本
- 按"行业 x 职能"矩阵逐步覆盖，每个场景复用底层 RAG/Agent 架构
- 场景包可作为方向 2（咨询培训）的后续转化产品
