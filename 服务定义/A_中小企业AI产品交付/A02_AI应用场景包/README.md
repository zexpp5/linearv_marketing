# A02 AI 应用场景包 / 行业解决方案

> 把技术栈封装成"解决某个具体岗位问题"的场景化产品。

## 产品线索引

### 通用场景产品（跨行业可卖）

| 编号 | 产品 | 一句话 |
|------|------|--------|
| 01 | 企业智能客服 | 接微信/企微/网站，AI 应答 + 人工兜底 |
| 02 | 销售线索助手 | 接 CRM，自动跟进/总结/评分 |
| 03 | 内部知识问答 | 接飞书/钉钉/语雀，员工即问即答 |
| 04 | 合同文档审阅助手 | 风险条款标注 + 模板对比 + 修改建议 |
| 05 | 电商运营助手 | 商品文案/客服话术/评论分析 |
| 06 | 招聘筛选助手 | 简历解析 + 岗位匹配 + 面试题生成 |

### 行业场景包（垂直深入）

| 编号 | 行业 | 覆盖场景 |
|------|------|----------|
| 07 | 电商/零售 | 选品、上架、舆情、售后工单 |
| 08 | 教育/培训 | 课程设计、学员答疑、效果评估 |
| 09 | 法律/财税 | 合同审查、政策问答、票据识别 |

| 13 | 通用职能效率 | 会议纪要、周报汇总、竞品监测、SOP 问答 |

## 商业模型

| 维度 | 说明 |
|------|------|
| 客户买单逻辑 | 解决一个具体岗位/部门的问题，看得见 ROI |
| 典型客单价 | 3-10 万 / 项目 |
| 收费模式 | 一次性交付费 + 年度运维费 |
| 交付难度 | 中 |
| 现金流速度 | 中（成交周期 2-8 周） |

## 落地建议

- 优先从 01-06 通用场景中选 2-3 个做成**可演示的标准 demo**
- 行业包（07-12）按客户需求逐步开拓，不急于全线铺开
- 13 通用效率包适合作为咨询培训（C02）后的**首单切入产品**
