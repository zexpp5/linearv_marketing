# A02 AI 应用场景包 — 5 人团队战略

> 不做产品竞争，做大厂做不了/不屑做的事。

## 核心判断

| 事实 | 来源 |
|------|------|
| 54% 大厂 AI 功能渗透率不足 10% | 53AI/SaaS 行业报告 |
| 60% 的 AI 项目会翻车 | 量子位 |
| 金蝶中小客户流失率 21% | 36 氪 |
| 企业"年耗 8 万上系统却仅用 20% 功能" | 证券时报 |

**结论：市场不缺 AI 产品，缺的是"帮你把 AI 用起来"的人。**

## 不做什么（已有成熟竞品占领）

智能客服（智齿/Udesk/美洽）、销售线索（探迹/纷享销客）、合同审阅（幂律/法大大）、
招聘筛选（Moka/北森）、医疗 AI（讯飞/森亿）— 这些赛道的竞品分析保留在 `_已有竞品参考/` 目录。

## 做什么（三条线）

```
01_AI实施与陪跑服务    → 帮客户把已买的 AI 工具用起来（B端）
02_微垂直行业AI工具    → 做大公司看不上的细分场景（B端）
03_跨系统AI集成        → 做企业系统之间的 AI 胶水层（B端）
04_C端个人AI服务       → 考试辅导/接单服务/付费咨询（C端，纯线上）
05_B端销售获客AI       → 招投标/渠道对接/Key Person BD（B端）
```

### 四条线的关系

```
C01 自媒体 IP（小红书/抖音发内容）
    ↓
04 C端个人服务（最快现金流 + 获客渠道）
    ↓
C02 咨询培训（获客入口）
    ↓ 诊断出客户需求
    ├→ 01 实施陪跑（客户已有工具，帮他用好）
    ├→ 02 微垂直工具（客户没有合适工具，我们做一个轻的）
    └→ 03 跨系统集成（客户有多个系统，帮他用 AI 串起来）
        ↓
    A05 托管运营（持续服务，月费收入）
```
