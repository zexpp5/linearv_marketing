# 制造/供应链行业场景包

> AI 辅助采购、质检、供应商管理，降低人工审核成本。

## 业务痛点

| 痛点 | 具体表现 |
|------|----------|
| 供应商审核靠人肉 | 资质文件几十页，人工逐条核验耗时且易遗漏 |
| 质检效率低 | 纸质质检报告手工录入，异常指标发现滞后 |
| 采购比价费时 | 多供应商报价格式不一，Excel 手工比对容易出错 |
| 数据散落 | 采购、质检、仓储各一套系统，决策缺乏整体视角 |
| 原材料价格波动 | 缺乏价格趋势预测，采购时机靠经验判断 |

## 场景清单

| 场景 | 说明 |
|------|------|
| 供应商资质自动审核 | 上传资料自动核验，风险点标注 |
| 质检报告自动解读与异常预警 | AI 读质检报告，异常指标实时告警 |
| 采购询价自动比对与汇总 | 多供应商报价自动对比，生成决策报告 |

## 目标客户

- 中小制造企业
- 供应链管理公司
- 采购部门（月询价量大）

### 我们的差异化机会

- **竞品问题**：SRM/MES 是重型平台，实施周期长、价格高，中小制造企业用不起；AI 质检偏硬件视觉，门槛高
- **我们的切入点**：
  - 不做 SRM/MES，聚焦**文档智能**：供应商资质审核、质检报告解读、询价比对
  - 轻量工具切入，部署快（2-3 周），价格低（3-8 万）
  - 本质是"给制造业做文档 AI"，复用底层 RAG/文档处理能力
