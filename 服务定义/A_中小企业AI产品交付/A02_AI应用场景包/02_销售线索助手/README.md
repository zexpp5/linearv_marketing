# 销售线索助手

> 接 CRM，自动跟进/总结，让销售把时间花在成交而不是整理上。

## 业务痛点

| 痛点 | 具体表现 |
|------|----------|
| 线索跟丢 | 销售同时跟 50+ 线索，20-30% 因遗忘而流失 |
| 客户信息靠手动查 | 每次跟进前花 15-30 分钟查工商信息、新闻、招聘动态 |
| CRM 填写率低 | 销售嫌麻烦不填，管理层看不到真实漏斗 |
| 跟进记录散落 | 微信、邮件、电话记录分散，交接时信息断层 |
| 话术靠经验 | 新销售没有好话术，老销售的经验无法复制 |
| 管理层无数据 | 每周靠销售自报周报，数据滞后且失真 |

## 核心功能

- CRM 数据自动清洗与线索评分
- 客户背景自动调研（工商信息、新闻、招聘动态）
- 跟进记录自动总结 + 下一步建议
- 邮件/微信跟进话术自动生成
- 销售周报/月报自动汇总

## 目标客户

- B2B 销售团队（5 人以上）
- 有 CRM 但利用率低的公司
- 销售线索多但跟进不过来的团队

### 我们的差异化机会

- **竞品问题**：探迹/纷享/销售易偏大客户，客单价高；卫瓴/尘锋聚焦私域，对传统 CRM 覆盖弱
- **我们的切入点**：
  - 轻量 AI 插件形式对接客户现有 CRM（不替换，而是增强）
  - 聚焦"跟进自动化 + 话术生成"两个最痛的点
  - 按效果说话：线索跟进率提升、销售人效提升可量化
