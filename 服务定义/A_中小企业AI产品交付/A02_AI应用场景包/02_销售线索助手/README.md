# 销售线索助手

> 接 CRM，自动跟进/总结，让销售把时间花在成交而不是整理上。

## 核心功能

- CRM 数据自动清洗与线索评分
- 客户背景自动调研（工商信息、新闻、招聘动态）
- 跟进记录自动总结 + 下一步建议
- 邮件/微信跟进话术自动生成
- 销售周报/月报自动汇总

## 目标客户

- B2B 销售团队（5 人以上）
- 有 CRM 但利用率低的公司
- 销售线索多但跟进不过来的团队

## 交付物

- CRM 集成插件或独立面板
- 线索评分模型（定制化）
- 话术模板库
- 销售数据看板

## 商业模型

| 维度 | 说明 |
|------|------|
| 客单价 | 3-6 万（部署）+ 2000-5000 元/月 |
| 交付周期 | 2-3 周 |
| 核心卖点 | 销售人效提升 30-50%，线索不再漏跟 |
