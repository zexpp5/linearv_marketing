# 02 微垂直行业 AI 工具

> 做大公司看不上的细分场景 — 市场太小不值得他们做，但对我们刚好。

## 选场景的标准

| 条件 | 说明 |
|------|------|
| 大厂不做 | 市场规模 < 1 亿，大公司 ROI 算不过来 |
| 需求真实 | 客户现在靠人工/Excel/微信群，效率极低 |
| 有合规或纠纷驱动 | 不只是"效率提升"，还有"不做会出事" |
| 交付轻 | 1-2 周能交付，微信小程序/企微 Bot 即可 |
| 可复制 | 同行业客户需求高度相似 |

## 横向规律

这些行业都卡在两个刚需触发点：**合规压力** 或 **客户纠纷**。

解法一致：替换微信群 + Excel，给老板一个"有记录、能查询、能提醒"的轻工具。

定价区间：月费 99-299 元（SaaS）或 2-5 万（定制部署）。

## 第一梯队：已验证 + 竞品空白明确

### A. 连锁门店 SOP 问答机器人（10-50 店）

| 维度 | 说明 |
|------|------|
| 痛点 | 店员流动大，新人不会操作/不知道流程，店长每天答重复问题 |
| 现状 | 纸质 SOP + 微信群问来问去 |
| 做什么 | 企微 Bot，店员拍照或语音提问即答 |
| 竞品 | 万店掌（4000+ 品牌，面向大连锁）、帷幄（依赖摄像头硬件，百店起）、InfiSight（中等规模） |
| **空白** | **无低门槛 SOP 问答机器人**，现有产品要么依赖硬件，要么需大规模实施 |
| 报价 | 2-5 万 + 2,000-3,000 元/月 |

### B. 财税代账公司 AI 工作台

| 维度 | 说明 |
|------|------|
| 痛点 | 一个会计管 30-50 家客户，月底加班严重，政策更新跟不上 |
| 现状 | 亿企赢/大账房/鲸算盘 做记账报税，但不管客户沟通和获客 |
| 做什么 | 政策问答 Bot + 客户沟通自动化（微信/企微）+ 工作提醒 |
| 竞品 | 亿企赢（充值制，数万起）、大账房（效率 4x）、慧算账、柠檬云（498 元/年） |
| **空白** | **客户沟通自动化为零**（与微信打通几乎空白）、**收入端 CRM 为零**（无获客/续费管理工具） |
| 报价 | 2-4 万 + 1,500-3,000 元/月 |

### E. 外贸工厂询盘回复与跟单

| 维度 | 说明 |
|------|------|
| 痛点 | 每天 10-30 封询盘邮件，手打英文回复，水平参差，报价靠 Excel |
| 现状 | 阿里国际站只服务平台内，独立询盘没人管 |
| 做什么 | 询盘邮件自动解析 + 多语言回复草稿 + 报价单填充 + 跟单提醒 |
| 竞品 | 小满 OKKI（中高端，月费数千）、孚盟 MX（ERP+CRM，重）、富通天下（规模化外贸）、外贸邦/贸小七（轻量，有免费版） |
| **空白** | **小工厂价格门槛高**（主流 CRM 月费数千）、**跟单/生产协同断层**（获客后到交货无人管）、**多平台碎片化**（1688/阿里国际站/WhatsApp/邮件割裂） |
| 报价 | 3-6 万 + 2,000-4,000 元/月 |

## 第二梯队：需求真实 + 大厂完全空白

### C. 装修/家装公司报价与客户管理

| 维度 | 说明 |
|------|------|
| 痛点 | Excel 做报价，客户跟进全在微信，历史记录找不到 |
| 做什么 | AI 辅助报价（户型/面积/风格 → 报价单）+ 客户跟进 + 聊天归档 |
| 竞品 | 酷家乐（设计类，非报价）、亮宅（全流程 SaaS）、装企通 ERP、悟空 CRM（开源免费）、哲迈云（报价分析，偏工程） |
| **空白** | **智能报价 AI 市场空白明显**（大多是模板报价，非真正 AI 自动生成）、**工具碎片化**（设计/CRM/供应链各自独立）、**5 人以下小公司低价方案不足** |
| 报价 | 2-5 万 + 2,000 元/月 |

### D. 招生/留学中介客户跟进

| 维度 | 说明 |
|------|------|
| 痛点 | 同时跟 50-100 客户，材料清单/签证流程靠记忆，漏跟丢单 |
| 做什么 | 客户状态追踪 + 材料清单自动生成 + 时间节点提醒 |
| 竞品 | 乾坤 AMS（50+ 模块，重）、申学邦（4500+ 机构，全流程）、留学 SaaS、申学纪、Zoho CRM（168 元/用户/月起） |
| **空白** | **小微机构（5 人以下）轻量工具几乎空白**、**AI 深度不足**（停留在提醒，缺智能推荐/材料审查/文书生成）、**定价不透明**（垂直产品均不公开价格） |
| 报价 | 2-4 万 + 1,500-2,500 元/月 |

### F. 物业/社区管理工单自动化

| 维度 | 说明 |
|------|------|
| 痛点 | 微信群接报修，工单散落，业主投诉回复慢，不知道处理到哪一步 |
| 做什么 | 企微报修 → AI 分类 + 派单 + 进度通知 + 满意度回访 |
| 竞品 | 极致物业（中大型为主）、明源云（大集团，实施成本高）、圣狐云（轻量但 AI 为零，≤2000 元/月）、有你智居（500-3000 元/月）、物业帮帮 |
| **空白** | **AI 工单能力下沉不足**（AI 语音报修/自动派单锁定在头部自研系统）、**租户沟通无闭环**（业主不知道工单进度）、**轻量产品无 AI**（圣狐云轻但没 AI，明源有 AI 但太重） |
| 报价 | 2-4 万 + 1,500-3,000 元/月 |

## 做法论

1. **先用 01（实施陪跑）接触客户**，在服务过程中发现垂直场景需求
2. **第一个客户免费/半价做**，换案例和口碑
3. **做完提炼成可复制模板**，第二个起正式收费
4. **同一垂直方向最多 3-5 个客户就验证**，决定是否继续
5. **优先选有"合规驱动"或"纠纷驱动"的场景** — 不是"锦上添花"而是"不做会出事"
