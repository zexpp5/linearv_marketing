# 方向 11：技术合伙人 / 联合孵化

> 不赚现金流，赚期权——用技术交付换股权/收入分成。

## 产品形态

- 按股权/收入分成换 AI 技术交付
- AI MVP 两周出 demo 的快交付
- 创始人 AI 技术顾问（按月）

## 商业模型

| 维度 | 说明 |
|------|------|
| 客户买单逻辑 | 早期团队没钱但有方向，用股权换技术合伙人 |
| 典型客单价 | 股权 5-20% / 收入分成 10-30% |
| 收费模式 | 股权 / Revenue Share / 象征性月费 |
| 交付难度 | 高 |
| 现金流速度 | 几乎为零（期权属性） |
| 战略价值 | 挑 1-2 个下注，一个命中顶前面所有业务 |
