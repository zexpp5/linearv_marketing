# 方向 3：定制开发服务

> 接住那些没法标品化的长尾需求，按项目报价。

## 产品形态

- 小程序/网页版 AI 工具开发
- 已有系统的 AI 能力改造（给老 CRM/ERP 加 AI）
- 企业微信/飞书/钉钉机器人开发

## 商业模型

| 维度 | 说明 |
|------|------|
| 客户买单逻辑 | 现有系统加 AI、老板想做一个"自己的 AI 工具" |
| 典型客单价 | 5-30 万 / 项目 |
| 收费模式 | 项目制一次性收费 + 后续维护 |
| 交付难度 | 高（需求多变、易扯皮） |
| 现金流速度 | 慢（周期 1-3 个月） |
| 战略价值 | 长尾收入来源，非主力但不能没有 |
