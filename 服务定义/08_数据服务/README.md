# 方向 8：数据服务（被严重低估的底座）

> 数据是 AI 项目做不好的 80% 原因，必须单独报价。

## 产品形态

- 企业文档清洗、标注、结构化
- 知识库从 0 到 1 的一次性搭建
- 非结构化数据资产化（合同/客诉/工单挖掘）
- 私有数据脱敏与向量化

## 商业模型

| 维度 | 说明 |
|------|------|
| 客户买单逻辑 | 想做 AI 但数据一团乱，必须先处理 |
| 典型客单价 | 5-30 万 / 项目 |
| 收费模式 | 项目制 + 后续迭代 |
| 交付难度 | 中高（人力密集） |
| 现金流速度 | 中 |
| 战略价值 | 所有 RAG/Agent 项目的前置必做环节，单独卖利润更好 |
