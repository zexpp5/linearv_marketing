# 方向 9：出海陪跑

> 国内团队用 AI 出海 + 海外工具进中国，两个方向都是窗口期。

## 产品形态

- 帮国内品牌/工厂用 AI 做跨境电商、独立站、TikTok Shop
- 帮出海团队做本地化内容、合规、支付接入
- 帮海外 AI 产品做中国代理 / 本地化

## 商业模型

| 维度 | 说明 |
|------|------|
| 客户买单逻辑 | 出海是业务刚需，AI 是加速器 |
| 典型客单价 | 5-50 万 / 项目（美元结算更常见） |
| 收费模式 | 项目制 + 分成 + 订阅 |
| 交付难度 | 高（跨文化、跨平台） |
| 现金流速度 | 中 |
| 战略价值 | 客单价比内贸高一档，美元结算，客户质量好 |
