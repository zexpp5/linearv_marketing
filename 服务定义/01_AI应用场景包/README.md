# 方向 1：AI 应用场景包 / 行业解决方案

> 把技术栈封装成"解决某个具体岗位问题"的场景化产品。

## 产品形态

- 企业智能客服（接微信/企微/网站）
- 销售线索助手（接 CRM，自动跟进/总结）
- 内部知识问答（接飞书/钉钉/语雀）
- 合同/文档审阅助手
- 电商运营助手（商品描述、客服话术、评论分析）
- 招聘筛选助手

## 商业模型

| 维度 | 说明 |
|------|------|
| 客户买单逻辑 | 解决一个具体岗位/部门的问题，看得见 ROI |
| 典型客单价 | 3-10 万 / 项目 |
| 收费模式 | 一次性交付费 + 年度运维费 |
| 交付难度 | 中 |
| 现金流速度 | 中（成交周期 2-8 周） |
