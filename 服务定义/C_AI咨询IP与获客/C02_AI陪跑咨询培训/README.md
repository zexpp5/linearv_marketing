# 方向 2：AI 陪跑 / 咨询与培训服务

> 中小企业不缺工具，缺"怎么用"，卖认知和落地方法论。

## 产品形态

- AI 落地咨询（老板 workshop、岗位梳理、ROI 评估）
- 团队 AI 培训（产品/运营/销售/程序员分岗培训）
- Prompt 工程陪跑
- AI 转型顾问（按月/按项目）

## 分职能培训框架

### 老板/管理层

| 模块 | 内容 |
|------|------|
| AI 认知重塑 | AI 能做什么/不能做什么，ROI 怎么算 |
| 战略沙盘 | 梳理企业哪些岗位可 AI 化，优先级排序 |
| 决策工具 | AI 选型评估模板、供应商比价框架 |
| 产出物 | 《企业 AI 落地路线图》+ 3 个月行动计划 |

### 产品/运营

| 模块 | 内容 |
|------|------|
| Prompt 实战 | 需求拆解 → Prompt 设计 → 效果评估闭环 |
| 内容生产 SOP | AI 辅助文案/海报/短视频的标准流程 |
| 数据分析加速 | 用 AI 做用户洞察、竞品分析、AB 测试总结 |
| 产出物 | 《岗位 AI 工具包》+ Prompt 模板库 |

### 销售/商务

| 模块 | 内容 |
|------|------|
| 客户画像生成 | AI 辅助线索清洗与客户背景调研 |
| 话术优化 | 用 AI 生成/迭代销售话术与邮件模板 |
| 跟进自动化 | CRM + AI 的自动跟进与总结 |
| 产出物 | 《销售 AI 工具箱》+ 话术 Prompt 库 |

### 程序员/技术团队

| 模块 | 内容 |
|------|------|
| AI 编程实战 | Cursor/Copilot/Claude Code 的深度使用 |
| RAG 架构入门 | 知识库搭建全流程（分块、向量化、检索优化） |
| Agent 开发 | 从单 Prompt 到多步 Agent 的设计与编排 |
| 产出物 | 《技术团队 AI 能力成长路径》+ 代码模板 |

### 客服/支持

| 模块 | 内容 |
|------|------|
| AI 辅助回复 | 知识库驱动的智能回复建议 |
| 工单分类 | 自动分类、优先级判定、情绪识别 |
| 质检自动化 | AI 抽检对话质量并生成改进建议 |
| 产出物 | 《客服 AI 升级手册》+ 话术模板 |

## 面向中小创业公司的模板框架

### 轻量咨询包（1-2 天）
- 创始人 1v1 AI 诊断（2h）
- 输出《AI 落地优先级矩阵》
- 推荐 3 个即刻可用的 AI 工具组合
- 报价：1-2 万

### 标准陪跑包（1 个月）
- 周度 workshop x 4 次
- 帮团队搭好 1 个核心 AI 工作流
- 输出岗位级 SOP + Prompt 库
- 报价：3-5 万

### 深度顾问包（按季度）
- 每月 2 次现场/远程辅导
- 持续迭代 AI 工作流 + 效果复盘
- 优先获得新场景包试用
- 报价：1-3 万/月

## 商业模型

| 维度 | 说明 |
|------|------|
| 客户买单逻辑 | 老板焦虑、团队不会用，需要专家"手把手带" |
| 典型客单价 | 内训 1-5 万/场；顾问 1-3 万/月 |
| 收费模式 | 按人天、按月、按项目 |
| 交付难度 | 低（重 IP 和方法论，轻技术） |
| 现金流速度 | 快（先收款后交付） |
| 战略价值 | 所有技术方案的前置漏斗，先咨询培训 → 顺势引出后续项目 |
