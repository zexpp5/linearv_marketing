# AI 应用场景包 — 大企业定制版

> 与中小企业版的区别：不交整套产品，交 AI 能力模块，嵌入客户现有 IT 体系。

## 交付形态

- API / SDK 形式交付，客户 IT 团队自行集成
- 对接客户现有系统（OA/ERP/CRM/工单系统）
- 私有化部署（客户自有服务器或私有云）

## 典型场景

- 内部知识库 — 对接客户已有文档系统，私有化部署
- 智能客服 — 嵌入客户现有客服系统，提供 AI 能力层
- 合同/文档审阅 — 对接客户法务系统或 OA 流程
- 数据分析助手 — 对接客户 BI 系统，自然语言查询

## 商业模型

| 维度 | 说明 |
|------|------|
| 决策人 | IT 负责人 / CTO |
| 典型客单价 | 30-100 万 / 项目 |
| 收费模式 | License + 实施费 + 年度维护费 |
| 交付难度 | 高（需对接客户 IT 体系） |
| 现金流速度 | 慢（决策周期 2-6 个月） |
