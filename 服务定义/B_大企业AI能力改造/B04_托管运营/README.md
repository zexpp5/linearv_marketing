# 托管运营 — 大企业版

> 大企业有 IT 但缺 AI 专项运维能力，需要外部专家持续支持。

## 与中小企业托管的区别

| 维度 | 中小企业 | 大企业 |
|------|---------|--------|
| 运维范围 | 整套系统我们管 | AI 能力层我们管，基础设施客户管 |
| 对接方式 | 直接操作 | 通过客户 IT 流程协作 |
| SLA 要求 | 灵活 | 严格（响应时间、可用性承诺） |

## 服务内容

- AI 模型效果监控与调优
- Prompt / Agent 持续迭代
- LLM 用量优化与费用审计
- 定期效果复盘报告

## 商业模型

| 维度 | 说明 |
|------|------|
| 典型客单价 | 2-10 万/月 |
| 收费模式 | 年度服务合同 |
| 交付难度 | 中 |
| 战略价值 | 深度绑定，持续收入 |
