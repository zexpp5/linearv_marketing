# 数据服务 — 大企业版

> 大企业数据量大、系统多、合规要求高，数据治理是 AI 改造的前置关卡。

## 与中小企业数据服务的区别

| 维度 | 中小企业 | 大企业 |
|------|---------|--------|
| 数据规模 | GB 级 | TB 级 |
| 系统数量 | 1-3 个 | 10+ 个 |
| 合规要求 | 低 | 高（数据分级、脱敏、审计） |
| 交付方式 | 我们处理完交付 | 需在客户环境内操作 |

## 典型项目

- 多系统数据打通与统一治理
- 企业级知识库搭建（对接多个内部系统）
- 大规模文档结构化（合同、工单、客诉历史数据）
- 数据脱敏与合规架构设计

## 商业模型

| 维度 | 说明 |
|------|------|
| 典型客单价 | 30-200 万 / 项目 |
| 收费模式 | 项目制 + 后续迭代维护 |
| 交付难度 | 高 |
