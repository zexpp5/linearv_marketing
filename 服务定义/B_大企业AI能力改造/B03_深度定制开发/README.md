# 深度定制开发 — 大企业版

> 帮有 IT 团队的大企业做 AI 能力嫁接，按项目深度定制。

## 与中小企业定制开发的区别

| 维度 | 中小企业（类别 A） | 大企业（类别 B） |
|------|-------------------|-----------------|
| 交付物 | 完整应用 | AI 能力模块 + 集成方案 |
| 对接方 | 老板/业务负责人 | IT 团队/架构师 |
| 技术要求 | 我们全栈搞定 | 需要配合客户技术栈和安全规范 |
| 项目周期 | 1-3 个月 | 3-12 个月 |

## 典型项目

- 给现有 ERP/CRM 加 AI 能力（智能推荐、自动摘要、异常检测）
- 企业内部 Agent 平台搭建
- 现有数据平台的 AI 升级（自然语言查询、自动报告生成）
- 业务流程 AI 自动化（审批、质检、风控）

## 商业模型

| 维度 | 说明 |
|------|------|
| 典型客单价 | 50-500 万 / 项目 |
| 收费模式 | 项目制（分阶段付款） + 后续维护 |
| 交付难度 | 高 |
| 现金流速度 | 慢（周期长，但单笔金额大） |
