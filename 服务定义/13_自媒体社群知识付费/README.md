# 方向 13：自媒体 / 社群 / 知识付费

> 不是正经业务，是前 12 条所有业务最便宜的获客漏斗。

## 产品形态

- 付费社群（3000-5000 元/年）
- 老板 AI 认知课
- 公众号 / 视频号 / 小红书 IP
- 行业 AI 报告售卖

## 商业模型

| 维度 | 说明 |
|------|------|
| 客户买单逻辑 | 信息差焦虑 + 圈层归属 + 低价试错 |
| 典型客单价 | 99-5000 元 |
| 收费模式 | 年费 / 单次 / 订阅 |
| 交付难度 | 低（重 IP、轻交付） |
| 现金流速度 | 快 |
| 战略价值 | 获客漏斗 + 品牌资产 + 个人 IP 放大器，60% 客户可能从这里来 |
