# 方向 10：AI 外包劳动力（AI-BPO）

> 不卖工具，卖"已经用 AI 武装过的人"，按人头/按结果付费。

## 产品形态

- AI 客服外包（一人做原来 5 人的工作量）
- AI SDR / 电销外包（线索 + AI 话术 + 人工成单）
- AI 招聘筛选外包
- AI 内容运营外包

## 商业模型

| 维度 | 说明 |
|------|------|
| 客户买单逻辑 | 老板听得懂"每月多少钱雇几个人"，听不懂"搭一套 Agent" |
| 典型客单价 | 5000-30000 元/月/人岗 |
| 收费模式 | 月费 / 按结果 / 按量 |
| 交付难度 | 中（需要 SOP + 管理团队） |
| 现金流速度 | 快 |
| 战略价值 | 把技术能力封装成劳动力，绕过"AI 太复杂"的决策障碍 |
