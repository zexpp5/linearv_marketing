## 一、注册公司

### 1.1 注册平台

北京企业服务e窗通：https://ect.scjgj.beijing.gov.cn（或下载"北京e窗通"APP）

### 1.2 需要准备的材料

| 材料 | 说明 |
|------|------|
| 公司名字 | 备2-3个，格式：`北京XX人工智能科技有限公司` |
| 注册地址 | 朝阳区地址+房屋编码（先解决地址再注册） |
| 法人身份证 | 你自己的，拍照上传 |
| 监事身份证 | 找个朋友，不能和法人同一人 |
| 股东身份证 | 一个人100%持股也行 |
| 手机号 | 法人+监事各一个，接验证码+刷脸 |
| 注册资本 | 建议100-300万（5年内实缴） |
| 持股比例 | 提前定好 |
| 北京e窗通APP | 法人和监事都要装 |

### 1.3 经营范围（在e窗通搜关键词勾选）

**必选：**
- 人工智能基础软件开发
- 人工智能应用软件开发
- 人工智能理论与算法软件开发
- 人工智能通用应用系统
- 技术开发、技术咨询、技术转让、技术推广、技术服务
- 计算机软件开发
- 数据处理和存储支持服务

**建议加上：**
- 信息系统集成服务
- 互联网数据服务
- 信息技术咨询服务
- 企业管理咨询
- 知识产权服务
- 销售自行开发的产品

### 1.4 注册流程

1. e窗通注册账号 → 法人身份证+手机号+人脸识别
2. 名称申报 → 系统自动查重
3. 填写设立信息 → 地址、资本、经营范围、股东、法人、监事
4. 电子签名 → 所有人在APP刷脸确认
5. 提交审核 → 1-3个工作日
6. 领营业执照 → 可邮寄到家

### 1.5 拿到执照后

| 顺序 | 事项 | 时间 | 费用 |
|------|------|------|------|
| 1 | 刻章（公章、财务章、法人章、发票章） | 1天 | 免费 |
| 2 | 银行开户 | 1-2周 | 0元（部分银行收管理费） |
| 3 | 税务报到（etax.beijing.chinatax.gov.cn） | 30天内 | 0元 |
| 4 | 社保/公积金开户 | 随时 | 0元 |
| 5 | 找代理记账 | 立即 | 200-500元/月 |

**纳税人类型选小规模**（税率低、记账简单、季度≤30万免增值税）。

### 1.6 注册地址方案

| 方案 | 费用 | 推荐度 |
|------|------|--------|
| AI智能体加速器（金辉大厦）免费地址 | 0元 | 先问能不能拿到 |
| 代理记账公司送地址 | 0元（绑记账200-500元/月） | 最省事 |
| 朝阳集中办公区虚拟地址 | 2000-4000元/年 | 集群地址，银行开户可能被卡 |
| 酒仙桥/将台实际租办公室 | 3000-8000元/月 | 能申请房租补贴50% |

### 1.7 AI公司后续资质

| 事项 | 什么时候办 |
|------|-----------|
| 软件著作权登记 | 有代码就登记（申请高新必备） |
| 算法备案 | AI产品上线对公众时 |
| 大模型备案 | 有自研大模型时（申请补贴加分） |
| ICP备案 | 有网站/小程序时 |

## 三、算力补贴详解

### 3.1 青年人才算力支持（门槛低，先报这个）

- **补贴**：合同金额的 **50%**，每年最高 **50万**
- **合同时间**：2025.1.1 - 2026.3.31 期间签的算力/模型/语料合同
- **计算方式**：已付全款按合同额，未付完按实际已付金额
- **条件**：
  - 青年人才创立的企业
  - 做大模型/行业模型训练优化/前沿技术开发
  - 有明确技术路线和算力需求规划
  - 资金只能用于算力采购、模型训练、数据标注
  - 合同双方不能是同一老板、不能是母子公司

### 3.2 北京市算力券（分四档）

| 档位 | 补贴比例 | 上限 | 适合谁 |
|------|---------|------|--------|
| 方向一 | 50% | 3000万 | 头部大模型企业 |
| 方向二 | 20% | 500万 | 有规模的AI企业 |
| 方向三 | 20% | 100万 | 开源项目（免申即享） |
| 方向四 | 10% | 30万 | 一般AI应用企业 |

**方向三免申即享流程**：把开源项目发到魔搭社区/开源中国 → 北京数字经济促进中心评审 → 平台帮你报 → 补贴直接打给你。

**申报入口**：北京市经信局官网 jxj.beijing.gov.cn

## 四、房租补贴（路线B）

### 4.1 申请条件

- 公司注册地在朝阳区
- 实际办公地在朝阳区
- AI方向业务
- 有正规租赁合同 + 租金发票
- 事后补贴（先付钱再报销）

### 4.2 补贴标准

- 实际租金的 **50%**
- 每年最高 **500万**
- 面积不超过 **2000平米**
- 连续不超过 **3年**

### 4.3 费用估算（酒仙桥/将台）

| 面积 | 月租 | 年租金 | 补贴50% | 实际成本 |
|------|------|--------|---------|---------|
| 50㎡（3-4人） | 3000-4000元 | 3.6-4.8万 | 1.8-2.4万 | **1.8-2.4万/年** |
| 100㎡（6-8人） | 6000-8000元 | 7.2-9.6万 | 3.6-4.8万 | **3.6-4.8万/年** |
| 200㎡（15人+） | 1.2-1.6万 | 14.4-19.2万 | 7.2-9.6万 | **7.2-9.6万/年** |

### 4.4 申请流程

1. 在朝阳区租办公室 → 签合同、付租金、拿发票
2. 关注朝阳区科信局申报通知
3. 提交材料（营业执照、租赁合同、发票、产权证明、企业说明）
4. 区里评审 → 公示 → 补贴打到公司账上
