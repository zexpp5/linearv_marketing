## 四、采集方式

### 方式一：手动搜索（最准确）

1. 在抖音搜索上述关键词
2. 找到对标账号
3. 截图或记录数据
4. 发给 Claude Code 整理到飞书表格

### 方式二：用蝉妈妈/飞瓜查（有现成数据）

1. 登录蝉妈妈 www.chanmama.com
2. 搜索账号名称或关键词
3. 可以看到完整的粉丝、点赞、爆款数据
4. 导出 Excel 发给 Claude Code

### 方式三：Coze 自动采集（自动化）

用扣子 Coze 搭建工作流，自动采集抖音视频数据并写入飞书表格。
参考教程：https://blog.csdn.net/m0_53539063/article/details/147285907
