# OBS Studio 快速上手指南

> 目标：用 OBS 录制「口播 + 录屏」的抖音竖屏视频
> 整理日期：2026-04-14

---

## 一、下载安装（5 分钟）

### Mac 安装

1. 打开 https://obsproject.com/download
2. 点击 **macOS** 图标，下载 .dmg 文件
3. 打开 .dmg，把 OBS Studio 拖到「应用程序」文件夹
4. 第一次打开时，系统会弹出权限请求：
   - **系统设置 → 隐私与安全性 → 屏幕录制** → 勾选 OBS
   - **系统设置 → 隐私与安全性 → 麦克风** → 勾选 OBS
   - **系统设置 → 隐私与安全性 → 摄像头** → 勾选 OBS
5. 重启 OBS，完成

---

## 二、第一次打开 — 自动配置向导

首次启动会弹出配置向导：

- 用途选择：选 **「只用于录制」**（不是直播）
- 分辨率：选 **1080x1920**（竖屏）或先默认
- 帧率：选 **30 FPS**（够用）
- 点击「应用设置」

---

## 三、录制参数设置（重要）

### 3.1 视频设置

菜单栏 → **OBS → 设置 → 视频**

| 参数 | 设置值 | 说明 |
|------|--------|------|
| 基础分辨率 | **1080x1920** | 抖音竖屏 |
| 输出分辨率 | **1080x1920** | 保持一致 |
| 帧率 | **30** | 口播不需要60帧 |

> 如果做横屏视频就设 **1920x1080**

### 3.2 输出设置

菜单栏 → **OBS → 设置 → 输出**

| 参数 | 设置值 | 说明 |
|------|--------|------|
| 输出模式 | **高级** | 更多选项 |
| 录像格式 | **MP4**（或 Muxed MP4） | 抖音兼容性最好 |
| 编码器 | **Apple VT H264 硬件编码器** | Mac 用这个 |
| 码率 | **6000 Kbps** | 画质清晰 |

### 3.3 音频设置

菜单栏 → **OBS → 设置 → 音频**

| 参数 | 设置值 |
|------|--------|
| 采样率 | 48kHz |
| 声道 | 单声道（口播够用） |

## 四、场景和源的概念（核心）

OBS 有两个核心概念：

- **场景（Scene）**= 一个画面布局方案（比如「纯录屏」「口播+录屏」）
- **源（Source）**= 场景里的每一层内容（摄像头、屏幕、图片等）

你可以理解为：**场景 = PPT 页面，源 = 页面上的元素**

---

## 五、设置三种常用场景

### 场景 1：纯口播（真人出镜）

1. 左下角「场景」→ 点 **+** → 命名「口播」
2. 「来源」→ 点 **+** → 选 **视频采集设备** → 选你的摄像头
3. 拖动调整画面大小，让人物居中
4. 「来源」→ 点 **+** → 选 **音频输入采集** → 选你的麦克风

### 场景 2：纯录屏（演示工具）

1. 「场景」→ 点 **+** → 命名「录屏」
2. 「来源」→ 点 **+** → 选 **macOS 屏幕采集**
3. 选择要录制的屏幕
4. 「来源」→ 点 **+** → 选 **音频输入采集** → 选麦克风

### 场景 3：口播 + 录屏（秋芝2046 同款，最常用）

1. 「场景」→ 点 **+** → 命名「口播+录屏」
2. 先添加 **macOS 屏幕采集**（全屏录屏，作为底层）
3. 再添加 **视频采集设备**（摄像头）
4. 把摄像头画面**缩小**，拖到**右下角或左下角**
5. 这样就实现了「大屏幕录屏 + 小窗口真人」的效果
6. 添加 **音频输入采集** → 麦克风

**效果：**
```
┌──────────────────────────┐
│                          │
│     屏幕录制内容          │
│     （演示AI工具）        │
│                          │
│                  ┌──────┐│
│                  │ 真人 ││
│                  │ 小窗 ││
│                  └──────┘│
└──────────────────────────┘
```

## 六、开始录制

1. 选好场景
2. 点右下角 **「开始录制」**
3. 正常讲解/演示
4. 讲完了点 **「停止录制」**
5. 视频自动保存到：**菜单 → 文件 → 显示录像** 查看

---

## 七、录制技巧

### 口播录制要点
- 眼睛看镜头，不要看屏幕
- 用提词器 APP（手机放摄像头旁边）
- 保持距离镜头 50-80cm
- 背景简洁，不要太杂乱

### 录屏演示要点
- 录之前把桌面整理干净，关掉无关窗口
- 字体调大（浏览器 Command + 放大到 125%-150%）
- 鼠标移动慢一点，观众能跟上
- 切换页面时停顿 1 秒

### 音频要点
- 录制前测试一下音量（OBS 底部的音量条应该在绿色到黄色之间）
- 安静环境录制，关窗关空调
- 如果有外接麦克风（哪怕 100 块的领夹麦），效果远好于电脑内置麦

## 八、快捷键设置（推荐）

菜单 → **OBS → 设置 → 快捷键**

| 操作 | 建议快捷键 |
|------|-----------|
| 开始录制 | Command + Shift + R |
| 停止录制 | Command + Shift + S |
| 切换到「口播」场景 | F1 |
| 切换到「录屏」场景 | F2 |
| 切换到「口播+录屏」场景 | F3 |

这样录制过程中可以随时切换场景，不用鼠标操作。

---

## 九、录完之后 → 剪映剪辑

OBS 只负责**录制原始素材**，后期剪辑用**剪映**：

1. 打开剪映专业版（免费）
2. 导入 OBS 录好的 MP4 文件
3. 剪掉口误、停顿、开头结尾的废话
4. 添加字幕（剪映可以自动识别语音生成字幕）
5. 添加背景音乐（音量调到 10%-20%，不要盖过人声）
6. 加上封面标题
7. 导出 → 上传抖音

## 十一、常见问题

### Q：录出来没声音？
→ 检查「来源」里有没有添加「音频输入采集」，并且选对了麦克风

### Q：画面卡顿？
→ 设置 → 输出 → 码率降到 4000Kbps，或者关掉其他占资源的软件

### Q：录出来是横屏？
→ 设置 → 视频 → 基础分辨率改成 1080x1920（宽x高）

### Q：文件太大？
→ 正常。1080p 录 10 分钟大约 500MB-1GB。剪辑导出后会压缩

### Q：Mac 录屏权限问题？
→ 系统设置 → 隐私与安全性 → 屏幕录制 → 确保 OBS 被勾选。改完需要重启 OBS
